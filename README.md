🎓 Udemy Clone Website

This is a Udemy Clone website created using HTML and basic CSS.
The project is focused on replicating the basic layout and design of the Udemy platform to practice front-end web development skills.

📄 Pages Included

Home Page – Displays courses, categories, and basic UI structure.

🛠️ Technologies Used

HTML

CSS

✨ Features

Clean and simple user interface

Course cards layout similar to Udemy

Responsive basic design

Beginner-friendly project

🎯 Purpose of the Project

This project was built as part of my learning journey to improve my understanding of HTML structure, CSS styling, and layout design by cloning a real-world website interface.
